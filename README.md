# Internet_Radio_ESP8266
This is an Internet Radio reciever made with ESP8266 on Arduino enviroment

Wiring of VS1053 board (SPI connected in a standard way) to Wemos D1 Mini
VS1053_CS   D8  (15)
VS1053_DCS  D0  (16)
VS1053_DREQ D3  (0)
MOSI        D7  (13)
MISO        D6  (12)
SCK         D5  (14)
XRST        RST
